# Argo CD ConfigMap Diff Viewer

A small Chrome extension for making ConfigMapGenerator diffs easier to read in Argo CD.

## Why

ConfigMaps generated by Kustomize `ConfigMapGenerator` are recreated when their content changes. In Argo CD, that often appears as a delete/create diff, which makes it hard to see what actually changed inside the `data` section.

This extension pairs the deleted and created ConfigMaps by their base name, then shows only the changed `data` keys:

- Added
- Removed
- Changed

## Usage

1. Install the extension from `extension/` as an unpacked Chrome extension.
2. Open an Argo CD application diff page.
3. Click the extension icon.

The diff viewer is injected only into the current tab when you click the icon.

## Packaging

For local development, load the `extension/` directory from `chrome://extensions`.

If you package it as a zip for distribution, zip the contents of `extension/` so that `manifest.json` is at the root of the zip.
